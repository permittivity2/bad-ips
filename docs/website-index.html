üìñ How It Works

  • Hunter servers monitor their own logs and block malicious IPs locally
  • Gatherer server collects blocked IPs from all hunters
  • IPs blocked on any server are automatically propagated to ALL servers
  • Built-in never-block CIDR filtering protects your trusted networks
  • Automatic expiration and cleanup (default: 8 days)

üõ†Ô∏è Supported Services

Out-of-the-box detection for:

  • SSH (sshd) - Brute force attacks
  • Postfix - Mail server attacks
  • Dovecot - IMAP/POP3 attacks
  • Nginx - Web server attacks
  • Apache - Web server attacks
  • BIND - DNS attacks

sudo apt update
sudo apt install bad-ips

üìö Documentation

After installation, configuration files are located at:

  • /usr/local/etc/badips.conf - Main configuration
  • /usr/local/etc/badips.d/*.conf - Detector configurations
  • /var/lib/bad_ips/bad_ips.sql - SQLite database

Useful commands:

# View service status
sudo systemctl status bad_ips.service

# View logs
sudo journalctl -u bad_ips.service -f

# List blocked IPs
sudo nft list set inet filter badipv4
